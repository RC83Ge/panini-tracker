EN BREF

Cette application est un outil personnel et gratuit pour suivre une collection d'autocollants Panini. Elle ne vend aucune donnée, n'affiche aucune publicité et ne partage rien avec des tiers à des fins commerciales.

1Qui sommes-nous

Panini FIFA World Cup 2026 Tracker est une application web non officielle, créée à titre personnel pour aider à suivre une collection d'autocollants. Elle n'est affiliée ni à Panini S.p.A. ni à la FIFA.

Le code source est hébergé sur GitHub Pages à l'adresse rc83ge.github.io/panini-tracker.

3Ce que nous ne collectons pas

  • Aucune information personnelle : pas de nom, email, ou numéro de téléphone n'est demandé.
  • Aucun compte utilisateur : l'application ne nécessite pas de connexion.
  • Aucun cookie de suivi publicitaire.
  • Aucune donnée de localisation.
  • Aucune transmission à des serveurs tiers à des fins commerciales ou publicitaires.

4Stockage local (localStorage)

Votre progression (liste des autocollants collectés) est enregistrée uniquement dans la mémoire locale de votre navigateur (localStorage). Cela signifie que :

  • Les données restent sur votre appareil et ne sont jamais envoyées à un serveur.
  • Si vous changez de navigateur ou d'appareil, vous ne retrouverez pas automatiquement votre collection — utilisez le bouton 💾 Sauvegarder pour exporter un fichier de secours.
  • Si vous videz le cache ou les données de navigation de votre navigateur, votre collection sera supprimée définitivement, sauf si vous avez une sauvegarde .json.

5Fichiers que vous exportez

Les fonctions Exporter PDF et Sauvegarder (.json) génèrent des fichiers directement dans votre navigateur, sans passer par un serveur externe. Ces fichiers sont téléchargés sur votre appareil et restent sous votre seul contrôle.

6Hébergement et journaux techniques

L'application est hébergée gratuitement via GitHub Pages. Comme tout hébergeur web, GitHub peut collecter des journaux techniques standards (adresse IP, type de navigateur, pages visitées) à des fins de sécurité et de performance. Ces journaux sont gérés par GitHub selon sa propre politique de confidentialité, disponible sur docs.github.com.

7Polices et ressources externes

L'application charge des polices d'écriture depuis Google Fonts pour l'affichage. Cela peut entraîner un échange technique minimal avec les serveurs de Google lors du chargement de la page, indépendant de toute saisie de votre part.

8Vos droits

Puisqu'aucune donnée personnelle n'est collectée ou stockée sur un serveur, il n'y a pas de compte ni de profil à supprimer auprès de nous. Pour effacer toutes vos données :

  • Utilisez le bouton 🗑 Reset dans l'application, ou
  • Videz les données de site de votre navigateur pour ce site, ou
  • Supprimez simplement les fichiers .json exportés sur votre appareil.

9Modifications de cette politique

Cette page peut être mise à jour si de nouvelles fonctionnalités modifient la manière dont les données sont traitées. La date de dernière mise à jour figure en haut de cette page.

10Contact

Pour toute question concernant cette politique, vous pouvez ouvrir une discussion sur le dépôt GitHub du projet.
